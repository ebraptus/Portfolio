Description

This project was just a simple project application, taking in arguments to make a command line interface to do simple actions and return certain results. This console app just included a help menu with a option to clear the console window, show all commands, and a rubiks cube scramble generator that I made just for the fun of it.

What I learned from making this game is

  • Learning C++ function pointers and how to better use memory to get addresses for functions and use them later
  • Dynamically using memory and garbage collecting
  • Input parameters given to programs to run it differently depending on what given
  • Using user input to gather information, and return expected results
  • Stopping exploits when it comes to allowing the person to type text into memory
