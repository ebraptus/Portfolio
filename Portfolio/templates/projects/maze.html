Languages: GameMaker Language

This game was made for my Vocational Tech class, made in game maker in colab with someone else, we together made the maze game where you have very little visibility by creating a mask and only creating "light" around the player and around torches around the map. This game was just for me to try and get used to masks and creating semi realistic lighting within a 2D game.

What I learned from making this game:

  • Better collision with players hitting walls
  • Using masks to create a lighting system
  • Creating trace rays for better lighting and collision
  • Gameplay mechanics and how they relate to the game as a whole and how to improve apon them
