Languages: TouchDevelopJS

This was the final project for my Computer Applications class, The class taught game programming through touch develop which is now a retired game engine my microsoft, but the game was our final project as we took our group member Kam and our teacher and made them fight to the death. The game was just a simple demonstration that I knew the basics of game programming and database integration.

What I learned from making this game:

  • Creating enemies and templating them so that they are indipendent but dont need seperate code
  • Creating enemies that shoot at the player and predict the players position and where they are going to move next
  • Networking behind making a database that holds all the leaderboard data and updates it with every game
